# nfd2nfc Homebrew Tap (Deprecated)

> **This tap is deprecated.** nfd2nfc is now available in [homebrew-core](https://formulae.brew.sh/formula/nfd2nfc).

## Migration

If you previously installed via this tap:

```bash
brew uninstall nfd2nfc
brew untap elgar328/nfd2nfc
brew install nfd2nfc
```

## Install (new users)

```bash
brew install nfd2nfc
```

For usage and documentation, see the [main repository](https://github.com/elgar328/nfd2nfc).
